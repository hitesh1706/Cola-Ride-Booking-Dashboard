# 🚖 Cola Ride Booking Dashboard

## About the Project

This project is a ride booking analysis dashboard created in Power BI using Excel data. The dashboard helps analyze overall business performance like bookings, revenue, cancellations, vehicle performance, and customer ratings.

The main goal of this project was to practice data analysis and dashboard creation by converting raw booking data into meaningful insights and easy-to-understand visuals.

---

## Tools Used

* Power BI
* Excel
* Power Query
* DAX

---

## Dashboard Pages

### 1. Overview Dashboard

This page gives a complete overview of booking performance.

It includes:

* Total Bookings
* Total Booking Value
* Total Customers
* Successful Bookings
* Cancellation Percentage
* Booking trend over time
* Top pickup locations
* Booking value by payment method and vehicle type

---

### 2. Vehicle Performance Dashboard

This dashboard focuses on vehicle category analysis.

Key analysis:

* Vehicle-wise bookings
* Successful bookings
* Success rate
* Average booking value
* Revenue generated by each vehicle type

Some findings:

* Prime Sedan generated the highest revenue
* E-Bike had the highest booking count
* Bike category had the highest success rate

---

### 3. Revenue Dashboard

This page analyzes revenue performance.

It includes:

* Total revenue
* Revenue trend by day
* Revenue by vehicle type
* Revenue by payment method
* Top pickup locations by revenue

Insights:

* Cash payment contributed the highest revenue
* Revenue remained stable across most days
* Some pickup locations generated significantly higher revenue

---

### 4. Cancellation Dashboard

This dashboard helps understand ride cancellation patterns.

It includes:

* Total cancelled rides
* Driver cancellations
* Customer cancellations
* Cancellation percentage
* Main cancellation reasons by Customer 

Insights:

* Driver cancellations were higher than customer cancellations
* Common driver cancellation reason was personal or car-related issues
* Major User cancellation reason was driver not moving toward pickup location

---

### 5. Ratings Dashboard

This page tracks customer and driver ratings.

It includes:

* Average customer rating
* Average driver rating
* Daily rating trends
* Vehicle-wise customer ratings
* Rating distribution

Insights:

* Average ratings stayed around 4
* Mini and Auto vehicles received slightly better ratings
* Ratings remained stable during the month
